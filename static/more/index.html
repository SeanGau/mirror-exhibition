或者工藝櫥窗《鏡子》

時間2022/4/15 (五) - 4/30 (六)
地點或者工藝櫥窗3樓
(新竹市新竹市中央路75號)
展出人王筱璦
林秀蘋（思良）/ BONTE好物金工
康雅筑 / 織物地圖
張博傑 / 真真鑲嵌玻璃研究所
黃虹毓 / 土星工作室
潘子村 / 村 碟樂
賴信佑 / 光山行漆器工藝

《鏡子》療癒工藝線上展呈現藝術家如何在工藝創作中汲取療癒力量。有別於以實體空間展出最終完成品的展覽形式，因應線上觀展特性，參展藝術家將工藝作品概念化，透過文字、影像和聲音呈現，引導觀眾一探帶來療癒的「創作過程」。

受或者工藝櫥窗邀請，規劃《鏡子》展覽番外篇，將播映衍生計畫《擁抱》，是結合舞蹈影像的紀錄片：以工藝類藝術家之創作過程出發，從聲動、觸動、時光、共生四子題切入，紀錄藝術家於創作過程中的轉折、因不同事件所產生的身體律動、感知及心境變化，並轉為舞蹈影像訴說。此外，現場也展出《鏡子》參與藝術家部分參展作品和其他系列的工藝作品，期待觀者不只透過線上展捕捉藝術創作的核心，也可親眼看到實體的質感。
